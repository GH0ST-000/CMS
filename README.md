## Set Up

To set up project, run the following command

```bash
 composer install
```
```bash
Copy .env.example to .env
```
```bash
 Run php artisan key:generate
```
```bash
 Run npm install & npm run dev
```
```bash
 Run php artisan migrate --seed
```
